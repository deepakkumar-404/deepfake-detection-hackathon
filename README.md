# Deepfake & AI Content Detection System

This project detects AI-generated and manipulated media (deepfakes) using machine learning models.

## Problem Statement
Deepfake and AI Content Detection

## Features
- Image-based deepfake detection
- Real vs Fake confidence score
- Web-based interface for easy use

## Tech Stack
- Python
- Streamlit
- OpenCV
- Deep Learning (Pretrained Models)

## Team
- Team of 4 members (Hackathon Project)

## Use Cases
- Misinformation detection
- Cybercrime investigation
- Media verification
